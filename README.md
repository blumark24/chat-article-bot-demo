# بوت إدارة الشات والمقالات

مشروع `chat-article-bot-demo` هو ديمو Web App بسيط يوضح فكرة بوت يعمل داخل غرفة دردشة تجريبية، يقرأ الرسائل، يرد تلقائياً من ملف أسئلة وأجوبة/مقالات، ويدير نظام نقاط وترتيب لاعبين.

## فكرة المشروع

- واجهة عربية RTL بالكامل.
- غرفة شات تجريبية بدون اتصال بأي موقع خارجي.
- بوت يمكن تشغيله أو إيقافه من لوحة التحكم.
- ملف أسئلة وأجوبة JSON داخلي مع إمكانية اللصق أو رفع ملف.
- إضافة سؤال وجواب يدوياً.
- نظام نقاط محفوظ في `localStorage`.
- سجلات تعرض وقت الرسالة، المستخدم، نص الرسالة، رد البوت، وهل تم احتساب نقطة.
- تصدير السجلات TXT، النقاط JSON، وإعدادات الأسئلة JSON.

## طريقة التشغيل

```bash
npm install
npm run dev
```

بعد تشغيل الأمر السابق افتح الرابط الذي يظهر في الطرفية، غالباً:

```bash
http://localhost:5173
```

## بناء نسخة الإنتاج

```bash
npm run build
```

ولمعاينة نسخة الإنتاج محلياً:

```bash
npm run preview
```

## ملاحظات مهمة

هذا المشروع Demo فقط:

- لا يوجد تسجيل دخول حقيقي.
- لا يوجد Backend معقد.
- لا يوجد اتصال بأي موقع شات خارجي.
- التخزين الحالي داخل المتصفح فقط باستخدام `localStorage`.
- لا يستخدم Playwright في هذه المرحلة.

## النشر على Vercel

المشروع مبني باستخدام React + Vite، لذلك يمكن نشره على Vercel مباشرة:

- Build Command: `npm run build`
- Output Directory: `dist`

## المرحلة القادمة

المرحلة القادمة ستكون ربطه بموقع الشات الحقيقي باستخدام:

- Node.js
- Playwright
- SQLite
- Windows Server

مع فصل منطق البوت، تخزين السجلات والنقاط في قاعدة بيانات، وإضافة طبقة تشغيل آمنة للمراقبة والردود.
